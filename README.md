# How to use my portfolio
1. Projects  
  1-1. 프로젝트 페이지에서 번호를 클릭하면 작업한 프로젝트를 볼 수 있습니다.  
  1-2. 프로젝트 이미지를 클릭하면 해당 github이나 youtube으로 연결됩니다.  

2. Skills  
  2-1. 해당 work와 관련해서 보유한 skill을 볼 수 있습니다.  
  2-2. View All을 클릭하면 한번에 모두 볼 수 있습니다.  

3. About Me  
  3-1. 각각의 알파벳에 맞는 제 이야기가 있습니다. 클릭해보세요:D  

4. Contact  
  4-1. 언제든지 빠르게 답장할 수 있는 이메일 주소가 있습니다.
